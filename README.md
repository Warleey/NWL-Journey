## 📌 Sobre o Projeto

O **NWL Journey** é uma aplicação web desenvolvida para treinar manipulação de datas, eventos e interações com JavaScript.

A proposta foi simular um checklist de tarefas para viagens, permitindo adicionar, marcar e gerenciar atividades de forma dinâmica.

## ⚙️ Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Manipulação dinâmica de elementos na interface
- Controle e formatação de datas
- Atualização da interface sem recarregamento de página

## 🎯 Conceitos Aplicados

- Manipulação de DOM
- Event Listeners
- Estruturação de lógica condicional
- Organização de funções
- Separação de responsabilidades no código

## 📱 Responsividade

A interface foi estruturada para manter boa usabilidade em diferentes tamanhos de tela.
